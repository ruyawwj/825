# v2ruya面具免流模块（v2ray，xray，免流）

使用双节点方法如下：
1.在模块目录找到"双节点开关.sh"执行
 显示"已开启双节点"
 
2.然后在文件夹"节点快捷切换"打开
 先去运行一个国内节点，再运行一个国外节点即可
